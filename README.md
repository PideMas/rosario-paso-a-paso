# El Santo Rosario versión Digital

Webapp estatica para rezar el Santo Rosario paso a paso.

## Publicacion

Este proyecto esta listo para publicarse en GitHub Pages o Cloudflare Pages.

Archivos principales:

- `index.html`
- `styles.css`
- `app.js`
- `manifest.json`
- `service-worker.js`
- `data/rosario.js`
- `assets/`

No requiere servidor ni dependencias externas.
